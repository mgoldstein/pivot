Documentation

Homepage Slider uses bxslider
Documentation can be found here http://bxslider.com/

USAGE
Create Homepage Slider nodes and publish them.
Once published add the nodes to the Homepage Slider Nodequeue.

To upload a new background image, configure the block by clicking the gear icon and add an image.
The image will get saved to /sites/default/files/hpslider directory

The slider is a block and uses context as of 10.23.14, this block will need to be added to the 'header' region in
the homepage context.
